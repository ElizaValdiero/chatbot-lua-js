# Chatbot Lua + JS

Projeto simples de chatbot com backend em Lua e interface web em HTML, CSS e JavaScript.

O sistema funciona como uma API local que recebe mensagens do frontend, processa regras básicas de FAQ e retorna uma resposta em JSON.

---

## Visão geral

- Frontend em HTML, CSS e JavaScript
- Backend em Lua (LuaSocket)
- Comunicação via HTTP local
- Respostas baseadas em palavras-chave
